# Features Implemented

The website includes the following features:

User Registration
User Login
Homepage with welcome message
Product page containing at least five grocery products
About Us page
Contact page with:
Email
Phone Number
Contact Form
Shopping Cart (Add-to-Cart functionality)
Responsive design for desktop and mobile devices

---

# Website Pages

## 1. Homepage
The homepage introduces Grab 'N Go and welcomes visitors to the online grocery store.

### Features
- Store logo
- Hero banner
- Welcome message
- Featured products
- Navigation menu

---

## 2. Product Page

The product page displays grocery products with:

- Product image
- Product name
- Price
- Short description
- Add to Cart button

Example products include:

1. Fresh Apples
2. Fresh Milk
3. Rice
4. Bread
5. Cooking Oil

---

## 3. About Page

The About page explains the purpose of Grab 'N Go.

### Mission

To provide customers with a convenient, affordable, and reliable online grocery shopping experience through quality products and fast delivery.

---

## 4. Contact Page

Customers can contact the store using:

- Contact Form
- Email Address
- Phone Number
- Store Address

---

## 5. Shopping Cart

The shopping cart allows customers to:

- Add products
- View selected items
- Update quantities
- Simulate checkout

During the development of this project, I encountered several challenges including:

- Choosing a suitable website template.

These challenges were solved by exploring Wix tools, practicing website customization, and testing the website on different devices.
